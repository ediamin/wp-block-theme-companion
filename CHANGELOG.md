# Changelog

All notable changes to this project are documented in this file.

## 0.1.0 - 2026-07-12

### Added

- Add CSS custom properties for `settings.layout.contentSize` and `settings.layout.wideSize`.
- Add WordPress core border-radius preset variables.
- Add a Theme.json Actions command for changing the active `theme.json` file.
- Show RGB values alongside hexadecimal values in color variable documentation.

### Changed

- Respect theme.json core preset toggle flags for color, duotone, font size, gradient, shadow, and spacing presets.
- Improve preset extraction type safety and guard malformed layout settings.
- Refresh development dependencies and test tooling.

### Security

- Update dependencies to resolve Dependabot alerts.

## 0.0.2

- Update logo.
- Update extension description in package.json.

## 0.0.1

- Initial release.
